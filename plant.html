Responsiveness Explanation:

When resizing the browser, the letters row breaks first. This happens because the letters use col-sm-2 classes (which stack below 576px), while the plant names use col-md classes (which stack below 768px). Since the sm breakpoint is smaller than the md breakpoint, the letters will stack into a vertical layout first as the screen gets smaller.
